# Registered Cars in Malaysia 2025
A Power BI data cleaning and data visualisation project on cars that were registered in Malaysia from January to November 2025. Made in attempt to answer questions about the most popular brands, fuel types, and months for car registry in Malaysia for the year 2025.

This repository contains the original data used, and the dashboard created using the dataset after some cleaning.

All documents within are free to be used as your own.

# Data
Data from https://data.gov.my/data-catalogue/registration_transactions_car
